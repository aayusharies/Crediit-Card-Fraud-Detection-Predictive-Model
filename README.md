# Crediit-Card-Fraud-Detection-Predictive-Model
End-to-end fraud detection on 284,807 real European credit card transactions (0.17% fraud rate, highly imbalanced). Python EDA with Pandas, Seaborn &amp; Plotly, plus benchmarking of 5 classifiers — Random Forest, AdaBoost, CatBoost, XGBoost, LightGBM — via scikit-learn. Best ROC-AUC: 0.946 (XGBoost).
